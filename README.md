# dir-text
read directory output text like this
<pre>
egg-example
├── app
│   ├── controller
│   │   └── home.js
│   └── router.js
├── config
│   └── config.default.js
└── package.json
</pre>

# use

```
  const dirText = require('dir-text');
  dirText(targetPath,ignoreDirs);
```
